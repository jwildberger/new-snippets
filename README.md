# New Snippets

Reusable snippet source files for Omni CMS-style authoring tables and their rendered output.

## Snippets

### Photo Carousel

Creates an accessible, progressively enhanced photo carousel.

| File | Purpose |
| --- | --- |
| `snippets/photo-carousel/css/photo-carousel.css` | Carousel styles copied from `https://www.westga.edu/_z_resources/css/photo-carousel.css`. |
| `snippets/photo-carousel/js/photo-carousel.js` | Carousel behavior copied from `https://www.westga.edu/_z_resources/js/photo-carousel.js`. |
| `snippets/photo-carousel/table.html` | Blank authoring table for inserting into the WYSIWYG editor. |
| `snippets/photo-carousel/omni-fragment.xsl` | Production XSL fragment with asset collection templates. |
| `snippets/photo-carousel/transform.xsl` | Standalone XSLT test harness that converts examples into final carousel markup. |
| `snippets/photo-carousel/examples/photo-carousel.table.html` | Example authoring table with real content filled in. |
| `snippets/photo-carousel/examples/photo-carousel.output.html` | Example transformed page. |
| `snippets/photo-carousel/examples/kennedy-chapel.table.html` | Kennedy Chapel source tables from the existing implementation. |
| `snippets/photo-carousel/examples/kennedy-chapel.output.html` | Kennedy Chapel transformed output generated by the standalone transform. |

The XSLT transforms tables with the `photo-carousel` class. Blank slide rows are ignored. Each slide row should include one image in the `carousel-image` cell and optional caption content in the `photo-carousel-caption` cell.

## Output Requirements

Pages using the rendered carousel need the CSS and JS assets loaded:

```html
<link rel="stylesheet" href="/_z_resources/css/photo-carousel.css" />
<script src="/_z_resources/js/photo-carousel.js" defer></script>
```
